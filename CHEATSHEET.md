# ⚡ 快捷键速查表

> 打印出来贴在显示器旁边！📌

---

## 🎯 变量操作（最常用！）

```
光标停在变量上       自动高亮所有相同变量 ✨
]]                  跳到下一个高亮
[[                  跳到上一个高亮
```

**示例**：
```python
price = 100        # ← 光标停这里，所有 price 自动高亮
total = price + 10 # ← 这个 price 也会高亮
return price       # ← 这个也会
```

## ✂️ 选择代码（复制/删除/修改）

### 选择函数/类
```
vaf    选择整个函数（包括 def）
vif    选择函数内部（不包括 def）
vac    选择整个类（包括 class）
vic    选择类内部（不包括 class）
```

**示例**：
```python
def hello():        # ← 光标在这个函数里
    print("hi")     # 按 vaf 会选中整个函数
    return True     # 按 vif 只选中这两行
```

### 选择参数
```
vaa    选择参数（包括逗号）
via    选择参数（不包括逗号）
```

**示例**：
```python
def calc(price, tax, discount):
#        ^^^^^
# vaa 选中: "price, "
# via 选中: "price"
```

---

## 🚀 快速跳转

```
]f     跳到下一个函数
[f     跳到上一个函数
]c     跳到下一个类
[c     跳到上一个类
```

---

## 🔄 参数交换

```
<Space>sa    当前参数和下一个交换位置
<Space>sA    当前参数和上一个交换位置
```

**示例**：
```python
def calc(price, tax):
#        ^^^^^
# 光标在 price 上，按 <Space>sa
# 结果：def calc(tax, price):
```

## 🐛 错误诊断

```
光标停留 2 秒        自动显示错误详情
<Space>e           手动显示当前行错误
]d                 跳到下一个错误
[d                 跳到上一个错误
<Space>q           显示所有错误列表
```

## 🎓 组合技巧

### 重命名变量
```
1. 光标移到变量上（自动高亮所有）
2. 按 ]]  → 跳到下一个
3. 按 ciw → 删除并输入新名字
4. 重复 2-3
```

### 复制函数
```
1. 光标在函数内
2. 按 vaf  → 选中整个函数
3. 按 y    → 复制
4. 按 p    → 粘贴
```

### 删除函数
```
1. 光标在函数内
2. 按 vaf  → 选中整个函数
3. 按 d    → 删除
```

### 修改参数顺序
```
1. 光标在参数上
2. 按 <Space>sa  → 往后移
   或 <Space>sA  → 往前移
```
